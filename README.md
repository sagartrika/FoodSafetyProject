# FoodSafetyProject

Food Safety Project is a Python project. It consists of a Jupyter Notebook (.ipynb) and relevant data (.zip).

This project explores restaurants in San Francisco, California, their health inspection scores, and changes in those scores over time.

## Usage

1. Clone this repository onto your machine.
2. Ensure that your machine has Python installed, along with the following packages: Pandas, NumPy, Matplotlib, Seaborn.
3. Read through the Jupyter Notebook (.ipynb) and run the Python cells.
